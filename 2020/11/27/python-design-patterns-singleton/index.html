Python 设计模式——单例模式

发表于 2020-11-27 | 分类于 Python | | 阅读次数:
字数统计: 4.6k | 阅读时长 ≈ 0:05

单例模式即确保类有且只有一个特定类型的对象，并提供全局访问点。因此通常用于日志记录、数据库操作、打印机后台处理程序等。这些程序在运行过程中只生成一个实例，避免对同一资源产生相互冲突的请求。

特点：

  • 确保类有且只有一个对象被创建
  • 为唯一对象提供访问点，令其可被全局访问
  • 控制共享资源的并行访问
