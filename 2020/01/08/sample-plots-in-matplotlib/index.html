Color

matplotlib 中定义颜色的方式有以下几种：

  • (R, G, B, A)，如 (1.0, 0.0, 0.0) 表示红色，第四项数字 A （可省略）表示透明度
  • 字符 b、g、r、c、m、y、k、w，分别表示蓝、绿、红、青、洋红、黄、黑、白
  • HTML 颜色字符串 #RRGGBB，如 #FFFFFF 表示纯白色
  • 灰度字符串，介于 0 和 1 之间的浮点数，如 0.75 表示中度浅灰
