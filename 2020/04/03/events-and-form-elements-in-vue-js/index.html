Form Elements

v-model 是 Vue.js 中用于 HTML 表单元素（input、select、textarea 等）的内置指示器。它能够在表单元素与数据之间创建双向绑定，使得不管数据怎样变更，元素的行为与数据值总可以保持一致性。

在上面的示例中，选中或取消 checkbox，在 input 中输入任意文本内容，与之关联的数据 dataValue 和 otherValue 的值都会同步发生改变。
